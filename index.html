Configure the firebase setup for login web version

Note

if you are already set up the firebase for then you can skip this step.

This step is very much important for web version login and register feature to work. So please follow the below steps carefully.

1. Create Firebase Project

3. Once logged in, you can see firebase console dashboard. Click on Add Project. Provide a project name, and then select country and then after that click on Create Project. For example, see the below image

9. Go to Project settings in firebase dashboard option.



10. Go to Project settings and scroll down you will find created web App there you find config option that will have firebase settings for front end.

11. here you have to set all details except the two fields 1. databaseURL & 2. measurementId. Here you can set only test word.



12. You need to set this details in eRestro admin panel -> Web Settings page from this Project settings page in firebase dashboard option

Deployment

Our e-shop web supports two types of deployment methods: Standard Build and VPS Hosting

1.Standard Build Method:- In this method,We are Make the preRender HTML which is directly put on server. In that all Page Data is fetched at Customer Browser.

1.VPS hoisting Method:- In this method,We Required Server which can run Node js application , in this Method we Put our code In server directory and run from Directly server which is increase SEO.

Initial Setup

Before deploying, configure your environment settings:

Method 1: Standard Build Deployment

Environment Configuration

.env file configuration:   
                                             NEXT_PUBLIC_SSR="false" // Standard Build Method [ Method 1 ]
  1. Automated FTP Deployment:
    1. Ensure the file ftp.js exists in the project root with the following contents: 
                                                              
                                                            var config = {
                                                                host: "xxxxxxxxxxxxxxxxxx",
                                                                user: "xxxxxxxxxxxxxxxxxxxx",
                                                                password: "xxxxxxxxxxxxxxxxxxx",
                                                                port: 21, // change this to 22 or anything else only if you are using SFTP or something else
                                                                localRoot: __dirname + "/build",
                                                                remoteRoot: "/",
                                                                include: ["*", ".htaccess"],
                                                                exclude: ["images/**"],
                                                                deleteRemote: false,
                                                          };
    2. Fill the File with details
    3. After Fill the details Run the following command to deploy: 
      npm run deploy
      • This will take around 5-10 minutes to deploy after this you see in terminal following message
    4. After this command you can see the this following message in your Terminal
    5. After Deployment check your web Domain:- http://YOUR_DOMAIN/
      • we have now Successfully deployed our web on server.

Note

If you are getting error that means there are something mismatch in Ftp details. for that case we can follow this following steps to deploy the web.

  1. Run npm run deploy command.
      2. After this command you will get dist name folder.
  2. open your server directory, in that open your web domain directory.
  3. In that directory Folder, Paste dist folder's All files and Folder. [ Manual Way without needing of FTP ]
    • Make sure only paste files & folder which are located in dist named folder , not whole project's files & Folder

Method 2: VPS Hosting Deployment

Environment Configuration

.env file configuration:   
                                             NEXT_PUBLIC_SSR="true" // VPS Method [ Method 2 ]
  1. Prerequisites:
    • VPS with root access
    • Ubuntu 20.04 or later
    • Node.js installed
    • Apache (Recommended)
    • PM2 process manager
  2. Server Preparation: 
    # Update system packages
                                                    sudo apt update
                                                    sudo apt upgrade -y
                                                    
                                                    # Install Node.js and npm
                                                    curl -fsSL https://deb.nodesource.com/setup_16.x | sudo -E bash -
                                                    sudo apt-get install -y nodejs
                                                    
                                                    # Install PM2 globally
                                                    sudo npm install -g pm2
  3. Project Deployment: 
    # Upload the eshop project folder to your VPS manually
                                # Example: Place it in /home/eshop directory
                                                
                                # Navigate to the project directory
                                cd /home/eshop
                                
                                # Install dependencies
                                npm install
                                
                                # Build the project (if applicable)
                                npm run build
                                
                                # Start with PM2
                                pm2 start npm --name "eshop" -- start
                                
                                # Ensure PM2 restarts automatically
                                pm2 startup
                                pm2 save

Troubleshooting

  • Check PM2 logs: pm2 logs eshop
  • Restart application: pm2 restart eshop
  • View running processes: pm2 list

Create your own Language

In our E-shop , we can Provide English | Hindi | Arabic default. If you want to Add more Language then you can also add your own language

For that Follow steps below:-

  1. Open the Folder Located in Public/locales
  2. Create your language's Own file , like Example:- For french use Name fr.json
  3. In that file, Fill the keyWords we have just update values not keys: [ see in this Arabic example ]
  4. After that open file i18.js
  5. in this above file see resources snipped , in that add your language as per other languages norm.
  6. After that Open File LanguageSwitcher.jsx which is located in src/components/LanguageSwitcher.jsx
  7. In that see languages Array , so add your language's object in that fill. 
    
                                    {
                                        code:"YOUR_LANGUAGE_CODE",
                                        countryCode:"YOUR_LANGUAGE_COUNTRY_CODE"  [ example:- french origin is France],
                                        name:"YOUR_LANGUAGE_NAME",
                                        flag:"FLAG_CODE"
                                    }
  8. save the files and see the results.
